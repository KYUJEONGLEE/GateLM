# GateLM Application (legacy reference)

This package preserves the previous Project/Application-scoped Chat demo as a compatibility and implementation reference. It is not the GateLM Tenant Chat client and is not included in the production-like public deployment.

Use the package-local `.env.example` and explicitly enable the legacy customer-demo API only for isolated local compatibility work. New product work belongs in `apps/chat-web` and `apps/chat-api`.
