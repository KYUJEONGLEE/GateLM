# Team Debate Notes

이 디렉토리는 GateLM 버전별 방향성, 역할 분담, 기술 선택을 팀원별로 토론하기 위한 working draft 공간입니다.

공식 구현 계약이나 release plan이 아닙니다. 합의된 내용만 이후 `docs/v*.*/contracts.md` 또는 `docs/v*.*/implementation-plan.md`로 승격합니다.

각 팀원은 버전별 자기 이름 디렉토리 아래에 의견, 반박, 근거 자료, 실험 계획을 Markdown 문서로 정리합니다.

- `v2.0.0/`: v2 목표와 v1.x 진행 방향 토론 공간
