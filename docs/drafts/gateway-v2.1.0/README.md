# Gateway v2.1.0 Draft

이 폴더는 future/draft material이다. 현재 `v0.1.0` release readiness나 Gateway v2.0.0 계약을 대체하지 않는다.

## 포함 내용

- self-host packaging과 installability 계획
- production image target 초안
- routing/evaluation dataset과 probe 초안
- acceptance matrix와 implementation task 초안

## 사용 규칙

- 현재 계약 판단은 `specs/gateway/v2.0.0/contracts.md`를 우선한다.
- 이 폴더의 `v2.1.0`은 software release number가 아니다.
- API, DB, Event, Metrics, Schema field로 승격하려면 별도 contract/spec 변경이 필요하다.
