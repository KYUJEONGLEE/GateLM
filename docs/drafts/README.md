# GateLM Draft Docs

이 폴더는 현재 release 계약이나 product release note로 확정되지 않은 future/draft 문서를 둔다.

Draft 문서는 설계 후보와 다음 track의 아이디어를 보존하기 위한 자료다. 현재 API, DB, Event, Metrics, Schema field의 근거로 바로 사용하지 않는다.

| 위치 | 의미 |
|---|---|
| `docs/drafts/gateway-v2.1.0/` | self-host packaging, advanced routing/evaluation, production image planning draft |
