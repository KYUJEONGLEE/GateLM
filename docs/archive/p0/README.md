# Archived P0 Documents

| Field | Value |
|---|---|
| Status | Archived |
| Applies to | GateLM 초기 3~5일 P0 구현 이력 |
| Active entrypoint | [`../../current/README.md`](../../current/README.md) |

이 폴더는 GateLM 초기 P0 구현을 위해 작성된 과거 문서 모음이다.

문서에는 작성 당시 경로인 `docs/p0/...`, v1 계약 우선순위, 과거 branch/role/scope 표현이 남아 있을 수 있다. 이는 당시 기록으로만 보존하며 새 작업의 계약, 역할, 범위, 우선순위를 판단하는 근거로 사용하지 않는다.
