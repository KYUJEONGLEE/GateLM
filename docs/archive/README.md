# GateLM Docs Archive

이 폴더는 현재 릴리즈와 계약의 Source Of Truth가 아닌 과거 문서를 보관한다.

현재 계약 기준은 `specs/gateway/v2.0.0/contracts.md`, schema, fixture다. 현재 릴리즈 준비 기준은 `docs/releases/v0.1.0.md`와 공개 문서다.

## 포함된 영역

| 위치 | 의미 |
|---|---|
| `docs/archive/v1.0.0/` | 제품 릴리즈처럼 보일 수 있었던 legacy milestone 문서 |
| `docs/archive/gateway-v2.0.0-planning/` | v2.0.0 구현 계획, PR packet, acceptance matrix, RC checklist 등 historical planning |
| `docs/archive/p0/` | 초기 P0 설계와 작업 기록 |

Archive 문서는 배경 이해, 의사결정 추적, 과거 fixture 확인에만 사용한다. 현재 계약/API/DB/Event/Metrics/security 판단과 충돌하면 archive 문서를 따르지 않는다.
