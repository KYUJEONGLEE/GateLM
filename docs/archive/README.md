# Docs Archive

| Field | Value |
|---|---|
| Status | Archived |
| Authority | 과거 결정과 실행 이력만 보존 |
| Active entrypoint | [`../current/README.md`](../current/README.md) |

이 폴더의 문서는 현재 구현 기준에서 제외된 과거 기록이다.

Archive 문서는 결정 이력, 과거 fixture, smoke 시나리오, 이전 구현 근거를 확인할 때만 참고한다. 파일 안에 `현재`, `source of truth`, `우선` 같은 표현이 남아 있어도 작성 당시의 문맥이며 현재 GateLM 기준을 선언하지 않는다.

현재 작업의 문서 상태와 읽기 순서는 [`../current/source-of-truth.md`](../current/source-of-truth.md)를 따른다.
